Key Mathematical Insights

What the algorithm learns:

  • Word embeddings (W_in): Dense representations capturing semantic similarity
  • Context mappings (W_out): How to predict context from embeddings
  • Distributional hypothesis: Words in similar contexts get similar embeddings

Computational complexity per training step:

  • Forward pass: O(D × |V|) for matrix multiplications
  • Softmax: O(|V|) for normalization
  • Backward pass: O(D × |V|) for gradients
  • Total per pair: O(D × |V|) - expensive for large vocabularies!
This is why optimizations like negative sampling are crucial!
